# TaskNest

A small collaborative task board (mini-Trello) built to test the
Solarch npm package as a backend, with a Next.js + Three.js frontend.

- **Backend**: Solarch (Node.js, SQLite, auth, realtime, file storage) — see `/backend`
- **Frontend**: Next.js + React + Tailwind CSS + Three.js — see `/frontend`

Full architecture, data model, and the day-by-day commit plan live in
`docs/PLAN.md`.
